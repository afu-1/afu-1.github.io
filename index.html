Projects

SDS 322E Wrangling and Visualizations

Using the large public dataset from the Austin government, I explored whether there was a significant difference of the ambulances that had been scheduled to come to work between the city of Austin and Travis County. This project was focused on tidying and wrangling data using dplyr functions.

SDS 322E Classification and Prediction

Continuing off of the Wrangling and Visualizations project, I investigated whether it can be predicted if the city of Austin or Travis county met the goal percentage of response time. As a result, this project focused on the classification using the factoextra and cluster packages. This project also focused on prediction of a binary variable with the assistance of the caret, plotROC, and ggcorrplot functions.
